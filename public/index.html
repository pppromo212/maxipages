Project Automation System

Automated setup for projects with Telegram bot interface

Welcome to the Project Automation System

This system provides automated setup and configuration for various projects. Use the Telegram bot to select a project, provide a domain name, and the system will automatically set up the project with all necessary dependencies, nginx configuration, SSL certificates, and PM2 process management.

Getting Started

To get started with the Project Automation System:

  1. Start a conversation with the Telegram bot
  2. Use the /setup command to start the setup process
  3. Select a project from the list
  4. Enter your domain name when prompted
  5. Confirm the setup to begin the automated process

The bot will provide progress updates as it:

  • Installs system dependencies
  • Detects and installs project dependencies
  • Configures nginx with your domain
  • Sets up SSL with certbot
  • Starts the project with PM2
